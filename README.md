# -
Выпусная квалификационная работа "Оценка потенциала ревитализации заброшенных зданий"
Автор - Забурунова Лилия Александровна
В данном репозитории представлена выпускная квалификационная работа магистра с разработкой метода оценки потенциала ревитализации заброшенных зданий, методикой, апробацией метода, исходными данными и результатами применения метода на реальных и изменённых данных
Работа посвящена проблеме заброшенных зданий в городах, проблеме фрагментарности информации о них и отсутствия чёткого представлений о потенциале их ревитализации. Разработанный метод оценки потенциала ревитализации, учитывающий как собственные характеристики здания, так и характеристики его окружения и места расположения, нацелен на решение этой проблемы через предоставление сравнимой оценки зданий.
Исходные данные для метода включают полигоны заброшенных зданий с их характеристиками (состояние здания, наличие статуса ОКН, является ли бывшее назначение гражданским, функциональная зона, этажность, сложность формы основания здания, расчётная площадь основания здания), точки городских сервисов (10 разных слоёв), линейный слой дорожной сети. 
Метод оценки потенциала ревитализации заброшенных зданий строится на интегральной оценке по самым значимым для ревитализации характеристикам зданий. Методология работы включает методы квалиметрии, эконометрики и статистики, геоинформационные методы и подходы к классификации и типологии заброшенных зданий. Метод включает следующие шаги: выявление заброшенных зданий, сбор и расчёт их характеристик; расчёт потенциала ревитализации и вспомогательного потенциала сноса; интерпретация результатов.
Структура репозитория: data - основная папка с исходными данными и таблицами с расчётами по Санкт-Петербургу и Ломоносову, а также с текстом работы.
